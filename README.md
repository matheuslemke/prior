# Prior

Prioritize your tasks. Know what is the best thing to do.

## Run Locally

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```

## Deployment

To preview the project run

```bash
  pnpm preview
```

To deploy this project run

```bash
  pnpm build
```

## Tech Stack

**Client:** React, Vite, pnpm

**Server:** Node
